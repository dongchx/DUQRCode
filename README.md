# DDQRCode

## Description

A QRCode Tool on iOS written by Objective-C

## Installation

DDQRCode is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'DUQRCode'
```
